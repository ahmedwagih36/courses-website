# Course Platform SPA

Simple single-page course catalog with smooth section switching, card animations, and a lightweight dark mode.

## Features
- Multi-section SPA navigation powered by vanilla JavaScript
- Featured and full course listings with subtle reveal animations
- Search box to filter courses by title
- Built-in dark mode toggle plus sticky header and fixed footer
- Responsive layout that adapts to mobile widths

## Tech Stack
- HTML5 for markup
- CSS3 for styling and transitions
- Vanilla JavaScript for SPA logic, dark mode, and filtering

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your browser (no build step required).

## Project Structure
- `index.html` – Core layout and sections (Home, Courses, Categories, About, Blog, Login, Contact, etc.)
- `style.css` – Global styles, animations, and responsive rules
- `script.js` – Navigation events, card animations, dark mode toggle, and search filtering
