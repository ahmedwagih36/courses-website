# Course Platform SPA

Simple single-page course catalog with smooth section switching, card animations, and a lightweight dark mode.

## Features
- Multi-section SPA navigation powered by vanilla JavaScript
- Featured and full course listings with subtle reveal animations
- Search box to filter courses by title
- Built-in dark mode toggle plus sticky header and fixed footer
- Responsive layout that adapts to mobile widths

## Tech Stack
- HTML5 for markup
- CSS3 for styling and transitions
- Vanilla JavaScript for SPA logic, dark mode, and filtering

## Getting Started
1. Clone or download this repository
2. Open `index.html` in your browser (no build step required)
3. That's it! Start exploring the courses

## Pages/Sections
- **Home** - Hero section with featured courses
- **Courses** - Browse all available courses with search functionality
- **Categories** - View course categories
- **About** - Learn more about the platform
- **Blog** - Read educational articles
- **Login** - User authentication form
- **Contact** - Get in touch with us

## Usage
- Click any navigation link to switch between sections
- Use the search box on the Courses page to filter courses
- Toggle dark mode using the button in the header
- All navigation is smooth with fade-in animations

## Browser Support
Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Project Structure
- `index.html` – Core layout and sections (Home, Courses, Categories, About, Blog, Login, Contact, etc.)
- `style.css` – Global styles, animations, and responsive rules
- `script.js` – Navigation events, card animations, dark mode toggle, and search filtering

## Customization
- Edit course content in `index.html`
- Modify colors and styles in `style.css`
- Add new features in `script.js`
